# Power BI Coffee Dashboard Project
In this project, I designed and developed an interactive Power BI dashboard for a coffee shop chain using transactional sales data. The dashboard provided an intuitive analytical experience, tracking key 3 business metrics such as revenue, transactions and quantities with product category performance across multiple store locations.  

## Key Contributions

### Dashboard Design & Development:
- Selected appropriate **chart types**:
  - Line charts for revenue and transaction trends over time with date hierarch.
  - Bar charts for product category and item comparisons.
  - Stacked visuals to highlight store-level and weekday/weekend performance.
- Incorporated **filters and slicers** for:
  - Time (month/day), store location, and product type.
  - Weekday vs. weekend analysis to compare consumer behavior.
- Applied a **consistent visual theme**:
  - Neutral background for clarity.
  - Distinct colors for product categories (coffee, tea, bakery, etc.) to improve readability.
- Highlighted insights such as:
  - Revenue and transaction peaks by month.
  - Store-specific sales performance.
  - Customer preferences across different product categories.
